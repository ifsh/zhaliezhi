014，好内容月报（202004）

Apr 30, 2020

订阅「好内容月报」：https://tinyletter.com/zhaliezhi

电影

  • ★★☆☆☆ / 2019 / 西班牙 / 《饥饿站台 El hoyo》 / (6.8/10)
    剧情甚至结尾都不算有新意，隐喻也有些浅薄。

  • ★★☆☆☆ / 2020 / 美国 / 《狩猎 The Hunt》 / (6.0/10)
    一部无聊的电影。

  • ★★☆☆☆ / 2012 / 中国 / 《箭士柳白猿》 / (6.5/10)
    大人，时代变了。

  • ★★★★★ / 2017 / 韩国 《1987：黎明到来的那一天》 / (9.0/10)
    真实最能打动人心，特别是经过这些运动之后韩国人大抵还是得到了那些靠血跟泪争取来的自由和民主。

  • ★★★★★ / 2018 / 韩国 / 《特工 공작》 / (9.0/10)
    有些细节还原得挺好的，不知道是在韩国搭景拍的还是后期合成的。黄政民第一次去平壤飞机空景俯瞰的时候远处有柳京饭店，顺安机场、大同江畔、金日成广场、平壤火车站等都还原得不错，真实度还是挺高的。

  • ★★★☆☆ / 2017 / 韩国 / 《铁雨 강철비》 / (6.9/10)
    在我看来这部电影远不如《特工》，总感觉剧情少了许多说服力。

  • ★★★★★ / 2011 / 韩国 / 《高地战 고지전》 / (9.2/10)
    比建国大业之类的不知道好的哪里去了。中国志愿军在山坳上冲锋那场戏真是精彩，音乐、火光、奔雷，一下子就出现的密密麻麻的人，真是让人头皮发麻又印象深刻。

  • ★★★★★ / 2005 / 韩国 / 《欢迎来到东莫村 웰컴 투 동막골》 / (8.9/10)
    野猪奔袭那场五分钟左右的戏真是太精彩了，升格画面把每个人的动作、神情都刻画了出来。当时就觉得这真是舞台剧的风格，后来一查才发现真是舞台剧改编的电影。

  • ★★★☆☆ / 2003 / 韩国 / 《实尾岛 실미도》 / (6.9/10)
    有些桥段还是觉得有些用力过猛，最后那段虽是「史实」，但其中的表现实在难以说是韩国最精锐的部队。

电视剧

  • ★★☆☆☆ / 2020 / 中国 / 《龙岭迷窟》
    看了前六集，可能以前鬼吹灯系列实在拍得太差了，所以这部稍微能看的就在豆瓣上得了高分。有几个用绿幕拍的场景实在太不走心了，处处是破绽，使得很难融入到剧情中。

播客

  • 09. 中国哭墙
    小哥斯拉蛋：政治吃了你，连个骨头都不吐。宁超文Victory：一束光照进铁塔，铁塔里的肮脏龌龊被显现，这束光便有了罪。不妨叫我洪老师：他们拿起一根棍子打了你，经过声势浩大的调查，结果是这根棍子受了处分。

  • 忽左忽右：我做讣闻记者的日子
    聊了一个比较冷门的话题：讣文。记得以前看《南方人物周刊》的时候，有一个固定的栏目叫「逝者」，大部分逝者都是普通人，文章作者或是逝者的亲人或是朋友，虽然不太记得具体的文章了，但对这个栏目印象还是挺深刻的。

  • 声东击西：#113 首相进了ICU，英国现在怎样了
    各国对抗疫情的政策反应了这个国家的政治、文化以及国民态度。

  • 灭茶苦茶：柳宗悦的活用
    「柳宗悦在一九二零年代思考什么是日本的文化身份，这也是今天的中国人思考的问题。民艺理论和大日本帝国时期（1868–1947）的民族主义脱不了干系，但它也是柳宗悦活用西方哲学、Arts & Crafts Movement、禅宗、茶道的集大成者。东方主义无罪，文化身份本来就必须经过人为的刻意构筑才会出现。」

  • 灭茶苦茶：CD 侧标论
    「唱片侧标、图书腰封，都是为了营销而存在、令不少人烦不胜烦的物件。但为什么竟有人会迷恋日本版唱片的侧标？除了收藏心态外，或许还有别的解释可能：包住是为了打开。」

  • 一天世界：Episode 79: 不要问互联网给了什么，问问它拿走了什么
    「居家隔离的时代，没有互联网怎么办？Social distancing，让许多人第一次过上了真正的互联网生活。原来在家工作这么难受？！」

  • 一天世界：Episode 75: 阔别二十年的电视
    「从一九九八到二零一八年我都没有看过电视并引以为荣。但随着互联网日益电视化，我们也有必要重新评估电视作为媒介对生活产生的影响——不好的方面被说得太多，今天我们来说好的方面。」

  • 硬影像：大乐必易：夜听黄霑
    「霑叔留给我们的遗产是丰富的，我感受更多的是他音乐传达出来的做人道理。」

  • 主场死了：第1期：为什么现在的后摇越来越同质化？
    以前从没有了解过后摇，可以在这个播客了解一些东西。这期推荐沼泽乐队的《争鸣》现场很棒，听完之后找来看了。

  • 字谈字畅：073：由繁入简易，由简入繁难
    「汉字的字形，随历史而演变。异体字的整理，汉字简化方案的提出及推行，以及各地区字形标准的差异，都为今日汉字的使用者和计算机软件的国际化，留下了不同程度的问题——当我们受惠于由繁入简的便利时，也不得不面对由简入繁的疑难。」

  • Anyway.FM 设计杂谈：看似重聊 Figma 使用体验，其实还是在吐槽 Sketch
    听了不少设计师聊 Figma，才知道原来 Web 也可以做得很好。

  • UX Coffee 设计咖：41 种蓝色
    「在这期节目里，我们和来自谷歌、Facebook、以及苹果的设计师和数据科学家，一起来探讨了这样一系列问题：数据，为什么成为了如今互联网公司的主宰？一个好的产品设计一定会从数字指标上反映出来吗？为什么有时候我们觉得更好的设计方案，产品指标却没有获得增长呢？作为身处数据驱动的公司员工个体，我们是否能找到一条行为准则来指导我们的工作？」

  • UX Coffee 设计咖：聊聊中文排版这件小事
    专业人士聊中文排版，可以得到不少参考。

  • 黃偉強·報導者：從政治力到民間力：台灣媽祖遶境的時代意義
    「大多數人的認知當中，宗教信仰應具有提倡道德、遠離政治、不談金錢這樣出世的價值觀，但事實上並非如此，民間信仰是多數台灣人生活的一部分，自然更與社會的政治經濟緊密結合在一起。」
