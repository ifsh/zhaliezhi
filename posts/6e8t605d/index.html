006，好内容月报（201908）

Aug 31, 2019

书

  • 老舍 / 译林出版社 / 《断魂枪集》
    一些短篇组成的小集子，印象深刻的有《断魂枪》，特别是最后写道沙子龙含混应对前来讨教的老者，关起门来全套断魂枪刺了下来，抚摸着枪身，微微一笑：「不传，不传。」还有《『火』车》开头的那一段描写。

  • 卡尔维诺 / 译林出版社 / 《命运交叉的城堡》
    把词组打散重新组合，有一千万种可能。

  • 钱莉芳 / 时代文艺出版社 / 《天命》
    感觉不如《天意》，这两本小说都是有了一个大框架，然后再寻找合理性和故事发展，人物更像是 NPC，跟着作者的叙事而动。

  • 何夕 / 人民邮电出版社 / 《何夕卷：爱别离》
    何夕的短篇作品合集，喜欢其中的《六道众生》、《爱别离》。

电影

  • ★★★★☆ / 2015 / 澳大利亚&美国 / 《疯狂的麦克斯4：狂暴之路》
    末世、废土、改装车、各种喷火，没有婆婆妈妈，不会停下来谈场恋爱，女人们不再是点缀，断了只手照样跟男人厮杀、拳拳到肉，这对我来说这才是顶级的爆米花电影，什么超级英雄、什么汽车人都弱爆了，如果在影院看 IMAX 那该有多爽啊。

  • ★★★★★ / 1995 / 香港 / 《大话西游之月光宝盒》
    没想到这么多年还是第一次完整地看完这部电影。

  • ★★☆☆☆ / 2018 / 韩国 / 《与神同行2：因与缘》
    第一部好歹还有些奇观化的场景展现，第二部更差了。

  • ★★★★★ / 2019 / 韩国 / 《寄生虫》
    奉俊昊的新片，得了戛纳金棕榈。有些细节处理还是比较粗糙，比如酒后的大厅，都是玻璃碴子、酒都洒在沙发上了，只是随便一扫，富人夫妇们灵敏的嗅觉竟然毫无动静；还有一家人的「寄生」实在太顺利了，如果宋康昊他们有如此精湛的驾驶技术和家务能力，基本上也很难找不到工作。最后，所谓的隐喻如果讲得太白了，回味起来就少了些韵味。

  • ★★☆☆☆ / 2018 / 日本 / 《来了》
    中岛哲也的新片，完全比不上《告白》，有一些奇观化的场景展现，其他很一般。

  • ★★★☆☆ / 2007 / 英国 / 《热血警探》
    剪辑凌厉、节奏明快，前半段比较好，后半段恶搞太过。

  • ★★★☆☆ / 2014 / 阿根廷&西班牙 / 《荒蛮故事》
    由六个独立的小故事组成，剧情并没有关联，「复仇」的内核倒是一脉相承。

  • ★★★★☆ / 2019 / 日本 / 《全裸导演》
    如果抛开故事情节，内核其实就是日本动漫一贯的青春、热血、奋斗，女性形象比真实故事发生过的表现得更好。

  • ★★★☆☆ / 2010 / 韩国 / 《孤胆特工》
    故事并不出彩，剪辑不错，其他一般。

播客

  • 路书：045.在日本寻找华夏之一：黄檗山万福寺
    「京都南郊的黄檗山万福寺是日本禅宗黄檗派的总本山，其开山祖师是清朝初年由福建赴日本传法的临济正宗第三十二世隐元隆琦。隐元不仅对日本禅宗有再造之功，其影响力还渗透至江户时代文学和艺术的许多方面。」

  • 天书广播：XXX：孟加拉
    孟加拉国，一个容易被忽略的国家。听了才知道孟加拉和孟加拉国有什么区别。

  • 疯投圈：36. 新消费品牌崛起大时代
    「品牌是消费类公司的核心竞争力。老品牌转型、升级的同时，新消费品牌也在衣食住行各个方面迅速萌芽，这是当下中国商业图景中最激动人心的大趋势之一。以苹果、戴森等为代表的「最佳产品」（Best Product）导向品牌将产品做到极致，甚至创造新的生活方式；小米等品牌主打「最佳价格」（Best Price），拥有极高的性价比；星巴克、耐克等品牌提供「最佳整体体验」（Best Overall Solution）实现口碑与市场双赢。黄海 与 Rio 从这三个类别讨论影响人们生活方方面面的消费品牌未来市场机会在哪里？」

  • 天书广播：LIII 缅甸
    缅甸虽然是邻国，但对其历史大多数人却并不熟悉，嘉宾的学识广博，很有趣。

故事

  • 黄国峻：失措
    「捂住、蒙起来、挡住；门、窗、棉被还有两手掌。能够阻隔得了的，都不是心所恐惧的。门内抵着餐桌，桌面上站着一张安乐椅，桌面下摆了一袋沙包。窗座的轨槽内——那虫蚁的坟场——填塞了旧毛巾，巾布饱吸了雨水，那拢闭的窗帘有什么作用？难不成风雨还会窥看屋里，然后决定要不要毁掉它，要多轻多重？」

演讲

  • 李显龙：国庆群众大会演讲
    李显龙的一个演讲，讲了新加坡开埠之后华人、马来人、印度人在此地相互融合，最终互相认同新加坡这个国家的一些历史。同时也谈到了新加坡作为一个小国，在中美两大国的较量下如何自处的经验。

信息条

  • 報導者：中國金主、菲國總部、台灣代工：看不見的線上博弈帝國
    「在短短幾年間，台灣已成為亞洲的線上博弈代工之島。
    《報導者》調查發現，即便政府尚未允許線上博弈產業，但過去3年，中國、香港、新加坡等資本前仆後繼來台，以遊戲設計、文字客服等名義設立公司，拿出高於市價2～5成的薪資，向30歲以下年輕人招手；在台灣低薪環境下，已有3萬名年輕工作者投入這個爭議和充滿投機的行業。這些博弈公司是如何登陸台灣，又怎麼吸收／吸引人才？
    不少年輕人更扛著膽識成為南漂移工，前赴菲律賓與柬埔寨「吃菠菜」（菠菜音似「博彩」）。他們怎麼進入、怎麼陷入？又如何看待自己的來時路和未來路？
    行業的超高風險和高利潤，已掀起島內人才轉移的風暴。透過網路，遠端服務大量的中國賭客，對這群低調的工程師、設計師、文字客服而言，是機會還是風險？
    亙古不變的賭性，如今透過台灣的人力人才，支撐起看不見的亞洲線上博弈帝國，政府不能再互踢皮球，不去正視這個隱藏在房間裡的巨象。」

  • 報導者：追求「無體溫」的親密關係：多元的陪伴文化與寂寞經濟正成形
    「台灣目前並沒有系統性、針對年輕人的親密關係研究。毫無疑問，與他人的連結與互動是構成健全人生的重要面向。在長工時、強調競爭力以及地位焦慮的年代，《報導者》試著從幾個點狀的現象探索，我們發現親密關係的維持如今成了一件奢侈品，「無體溫」（不強調真實生命連結、甚至不曾碰面）的親密關係正在浮現，包括：
    鐘點的出租情人、90秒的私密攝影會、各式線上社交軟體的大量使用⋯⋯網路原生世代發展出創意與多元的「陪伴文化」，滿足身體和心理的需求。
    這些浮雲般的親密關係背後，是怎麼樣的心情與投射？對於浪漫愛的想像和實踐，可以有什麼變化？進入穩定的關係，是否不再是人生必經的歷程？這個系列沒有試圖給出答案，因為一切都在進行式，讓我們一起探索下去⋯⋯」

  • 報導者：台灣人在中國：生錯時代的博士教授
    2016 年的一个专题报道，三篇文章加一篇摄影师的图文。台湾自 1994 年教改之后，产出的博士与年俱增，市场消化不了，于是一些到了美国、香港、新加坡，近些年更多的博士们「西进」到了中国。到了中国之后，一般有两个方向，去知名的大学，学术氛围好，但钱比较少；去不三四线的城市、大学，拿的钱会比较多。同时这群人也面对着完全迥异的学术升迁规则、政治氛围，最后，有的人留下，有的人还是选择回到台湾。

  • 纽约时报中文网：站在“十字路口”，范冰冰谈丑闻与复出
    范冰冰说，她已经为下一幕做好了准备，不管接下来会发生什么。「可能会有缺失会有遗憾，会有一些伤痛或者是脆弱，」她说。「但是我还是觉得要活下去。」

  • 纽约时报中文网：被侮辱的与被损害的
    法尔昆达，阿富汗的一名年轻女性，被控在一座清真寺前焚烧《古兰经》，随后被数十名男子经过几小时的折磨致死。这件事在阿富汗引起轩然大波，特别是在查证法尔昆达并没有焚烧《古兰经》之后。其后各方势力不断交锋，企图在这个事件中获得利益打击对手，从而开启了一场仓促的审判。

  • FT中文网：《美国工厂》：美国解读VS中国解读
    「整体而言，我不认为《美国工厂》试图“抹黑”曹德旺和福耀玻璃，反而想要借着一个比较中立和人性的角度，来看现实经济生活中中国人和美国人的相遇。不管曹德旺喜不喜欢这样的结果，我个人认为我们应该不要以为只有歌功颂德才是正能量，或代表中国的正面形象。借由艺术刻画的复杂人性，反而更能够让国际能够了解中国和中国人。」
