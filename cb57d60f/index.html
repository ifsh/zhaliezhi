011，好内容月报（202001）

Jan 31, 2020

电影

  • ★★★★★ / 1997 / 意大利 / 罗伯托·贝尼尼 / 《美丽人生 La vita è bella》
    好电影。开头的一些情节如果以二十几年后来看其实挺不合时宜的，不过还是演出了人在大时代下的抉择与挣扎。

  • ★★★☆☆ / 2017 / 中国 / 王冉 / 《闪光少女》
    唢呐一出，非死即伤。电影可能更适合青少年观看，情节、表演都没什么出彩之处，唯有那种青春年少的感觉出来了，还是青少年演青少年比较好，那种几十岁还靠着修图演十几岁的实在看不下去。

  • ★★★☆☆ / 2014 / 德国 / 巴伦·博·欧达尔 / 《我是谁：没有绝对安全的系统 Who Am I - Kein System ist sicher》
    很多时候系统当中最大的缺陷是人，只要有人参与就有漏洞。

  • ★★★★★ / 1999 / 香港 / 杜琪峰 / 《枪火》
    短小精悍、干净利落，值得看很多遍。

  • ★★★★★ / 2016 / 香港 / 许学文、欧文杰、黄伟杰 / 《树大招风》
    电影片长九十七分钟，讲述九七回归前夕的三大贼王。苦心寻觅却不见，回头望去不经意间却在风满楼相遇，其实电影想讲的是香港为什么是香港。

  • ★★★☆☆ / 2001 / 美国 / 亚历杭德罗·阿梅纳瓦尔 / 《小岛惊魂 The Others》
    如果十几年前看这部电影应该会觉得很惊喜，现在看的时候大致已经能猜出结局了。

视频

  • 吴坤明：宇宙快递 Johnny Express
    快递飞船蛮横的入侵，小人国里的各种标识，隐喻呼之欲出。

播客

  • 天书广播：朝鲜三国
    高句丽、百济、新罗都是不陌生的名字，但大多数人对朝鲜三国时代的历史所知不多，其对朝鲜半岛以及东北的影响却时至今日。

  • 社会人科技评论：Episode 17: 【特别制作】来去满洲，梦碎新京
    「本期节目是社会人科技评论的特别篇，采用了「故事叙事」的形式进行制作。并且与视频节目《有意思吗YOUYISIMA》音画联动，一起探究一段八十年前的故事。
    「来去满洲」曾经是抗日战争前不少日本内地与殖民地人的口头禅，在他们的心目：新满洲就是新天地。但这新天地又是建立什么基础上呢？而这所谓的「新天地」，又真能满足各个心怀鬼胎的人的梦想吗？从溥仪到烧锅作坊杂工；从穷苦学生到试图开拓满洲的日本普通人——本期的社会人科技评论，试图用音频从侧面复现这段属于个人的历史。」

  • 日谈公园：南锣鼓巷消亡史
    虽然我从没有去过北京，也没有去过南锣鼓巷，但听这期节目还是有些感动。

  • 社会人科技评论：Episode 18: 山东酒局的鲁棒性与建构地域/性别矛盾
    好客山东欢迎你。

信息条

  • 陳牧民·报道者：伊拉克庫德斯坦自治區實地觀察：混亂世界的淨土，會是獨立建國的希望？
    「自治區的主要城市如艾比爾與蘇萊曼尼亞（Sulaymaniyah）也都相當現代化，可以看到各式餐廳、飯店、購物中心等，茶攤裡坐滿悠閒喝茶聊天的人們，看起來已經從兩年前的經濟制裁中恢復，不過似乎並無太多外來觀光客造訪。」

  • 何欣洁·端传媒：成为马祖：大选前夕，战争凝固的岛屿
    「六十年后，曾一度远去的战意，似乎又隐约重临台湾海峡。在台湾大选前夕，我们重返两岸对峙的地理前线马祖，这个因国共内战、韩战而生，被旧冷战时代塑造的小岛。半世纪以来，在一场还未真正到来、但也没有离开的战争中，马祖如何永远地被改变？生活在『等待战争』之中，是怎样的滋味？」

  • 陳薇安：廉價女孩的代名詞，那些被稱為「騷底」的台妹
    「稱呼一個人『台妹』時，說話者間接把自己放在觀看者，甚至消費者的位置。台妹是沒有想法的，她教育程度不高，敢玩，不懂品味，對男人來說，甚至不是情感幻想的對象，僅僅只是男性情色的玩物。當然，我們要說的並不是擁有這些特質的女性比較『低等』，而是這個詞彙背後蘊含的權力觀。」

  • 沙青青·澎湃新闻：苏联、台湾交往秘辛
    一段少有人知的历史，中苏交恶，中美走在建交的边缘，苏联便找上台湾，希望能给予中国一些压力。蒋介石盼望能借苏联力量反攻大陆，最终在变幻莫测的国际局势下黯然作罢。

  • 劉怡·端傳媒：北敘利亞戰地手記：羅賈瓦，被入侵的烏托邦
    「在被土耳其入侵的北敘利亞庫爾德「羅賈瓦」政權裏，那些追求烏托邦的外來者終究會離去，北敘利亞住民們則期盼回到一種可忍受的日常秩序。他們擔心被世界遺忘，卻並不指望成為地球的焦點。」

  • 楊宗翰／空屋筆記：我在克羅埃西亞翻垃圾桶：完整的牛奶、優格、餅乾──幾乎沒有一樣是過期的
    「然而，這樣子的政策絕對不是一夕之間蹦出來的，政府之所以願意去處理剩食問題，是因為政府意識到了民眾在意這件事情，而民眾之所以會在意這件事，正是因為法國當地無數位 Dumpster diver 不斷地犯法、不斷地衝到垃圾桶裡面拍照，將一批一批的可以食用的垃圾拿出來展覽，公諸於世，不斷告訴民眾食物浪費的問題，才漸漸改變民眾的思維的。」

  • 纽约时报中文网：逃离马加丹，逃离极寒和古拉格的阴影
    「1994年，在佛蒙特州度过多年流亡生活的索尔仁尼琴返回俄罗斯，他从阿拉斯加出发，将马加丹作为他的第一站，抵达后他解释说他『是来向这片成千上万同胞被处决、埋葬的土地鞠躬致敬』。即使在今天，居民仍然将俄罗斯其他地区称为『大陆』，说明这座城市感到多么地孤立。为了振作精神，公交车上醒目地展示着对城市生存意愿的抗争声明：『马加丹有曾经，有现在，也有未来。』」

  • 庄舟：芥川龙之介的上海行，幻想的破灭
    「芥川龙之介难掩失望。『现代中国有什么呢？政治、学问、经济、艺术，难道不是悉数堕落了吗？特别是艺术，嘉庆道光以来，有一件值得夸耀的作品吗？国民不分老幼却都在高唱太平盛世。诚然，在年轻国民当中也许能看到一点活力。然而，他们的声音纵然能在全体国民胸中回响，事实上也缺乏巨大的热情。我不爱中国了，即使想爱也不能爱。』他眼中的城隍庙是病态的绿色池塘，『那隆隆作响、斜着注入池塘的一股尿，是帝国没落的辛辣可怖的象征。』他从前读中国小说，看到不少浪荡子或神仙装扮成乞丐的样子，但现实看到的中国乞丐，『却浪漫的叫人退避三舍，膝盖上的肉腐烂得像剥开的石榴。』」

  • 季末：别告诉她，别骂她
    「然而，一看到亚裔演员就觉得丑，就觉得受到歧视，就觉得审美霸权。这到底是指出了西方审美霸权的真相，还是我们把自己的审美强加于所有演员，忽略专业能力，武断地把主观判断凌驾于专业评审之上，占据道德高地？看到华裔就嫌丑，到底是谁的审美霸权？」

一首词

金門高粱
作者：白靈

只有砲火蒸餾過的酒
特別清醒
每一滴都會讓你的舌尖
舔到刺刀

入了喉，劃作一行驚人的火
燙進了歷史的胃袋
有誰的脖子和耳根
不紛紛升起
金門的輝煌
和悲涼

整片台灣海峽唯這座島
配做肚臍眼
半世紀的駭浪驚濤
都裝在裡頭
要幾瓶酒才倒得光

始終倒不出來的是歲月吧
從空酒瓶口望進去
望遠鏡中
卻是沒有一條船穿得透的
茫茫濃霧

那就趁半醉半醒
雙手朝兩頭一推
把海峽兩岸都推到
千年之外

但此時你卻醒在
酒瓶堆上
揉揉眼睛說：
「天啊，這裡種下的砲彈
竟比長出的高粱還多」
