Jun 30, 2019

订阅「好内容月报」：https://tinyletter.com/zhaliezhi

电影

  • ★★★★☆ / 2016 / 中国 / 《大护法》
    暴力、谎言和恐吓是欧阳吉安统治的手段，行刑者得到隐婆的唤醒，点燃革命的火种，推翻残暴的独裁者，却也把愤怒投向不够革命、拒绝摘掉假眼睛的花生人们。革命者自我觉醒推翻独裁者，又用独裁者的手段对付不够革命的同伴，慢慢成为新的独裁者。

  • ★★★★★ / 2017 / 台湾 / 《大佛普拉斯》、《血观音》
    几个细节：1. 同样是一个警察在查真相，大佛里面就直接用权势压迫他，让他知道真相也不敢不能说出来；《血观音》里面又是色诱，又是艳照，从结构性方面来说更强，但在现实性方面却是弱的。2. 两个在警局出来了之后，在肚财房子外面吃便当，也没有让他进去。最后再从菜埔的拜访中交代，他有太空舱、一房间的娃娃，告诉我们肚财不是一个淹没在人群中的符号，他也是跟我们一样的人，活生生的人、有血有肉。3. 看到杀人事件后，两人都没想过报警，肚财说法院跟警局都是有钱人开的。结合他之前在路上无缘无故被抓、又被放的情节，一切的发生没有理由、也没有道歉，最后他只得到了两个便当。
    在 2014 年金马最佳短片《大佛》也有几个细节：两人目睹了行车记录仪里面的杀人事件，肚才要走，在工厂门口跨上摩托车的时候，犹豫了一下，下车，不点火，推着走了。还有一场戏是俩人求心安是给工厂里的大佛拜拜，刚开始说台语，肚财提醒这个小三来自上海后菜埔开始说国语，说着说着又换回台语了。

  • ★★★★★ / 2014 / 中国 / 《心迷宫》
    一副棺材，两条人命，就像潮水退去后裸露在外的粗粝岩石，直挺挺地暴露了中国最深切的农村现状。那些灰暗的、隐秘的、不可言说的东西，在导演的镜头下像一幅被墨汁晕开的白纸，虽然不激烈，也没有一针见血，但就是无可抑止地流进每一个人的心里，留下了难以抹灭的痕迹。

  • ★★★★★ / 2017 / 韩国 / 《出租车司机》
    从来不需要想起，永远也不会忘记。

  • ☆☆☆☆☆ / 2019 / 中国 / 《秦明·生死语者》
    离及格还有很大一段距离，这两三年都没有看过这么烂的电影了，一个值得夸的点都没有。

播客

  • 疯投圈：28. 学习爱好者最好的时代
    嘉宾关于知识付费的一个比喻：「知识付费就像办了一张健身卡，请了一个私教，把各种动作一一分解给你看，但练不练是你的事情；教育的目的是功夫长在身上，追求认知跟身体能力同步，既知道要怎么练，也让你去练了，最终达到一个进步。订了很多知识付费，一一听完好像都懂了，这是『认知虚胖』，脑子比身体大，而教育是脑子跟身体同时进步。」

  • 反派影评：77.《出租车司机》：我们不需要想起，永远也不会忘记
    看完电影之后配合着听。

  • 不可理论：24. 阅读项飚
    还没听完就买了项飚的书《跨越边界的社区》，有些新的启发和反思。

  • 肖文杰·第一财经YiMagazine：中文播客的进化之路
    「在采访中，有不少播客制作者都提到，自己最早听的一个“中文播客节目”是《锵锵三人行》的音频版。说得直白点，中文广播电台不但未能向播客行业输送太多人才，甚至连可供借鉴参考的节目风格和理念都不多。」

  • 張育軒·報導者：為何曾經輝煌的阿拉伯，無力掌握今日命運？
    「是什麼讓你覺得我們是可以被玩弄的？因為我們是弱小的族群？愚昧的族群？貪婪、野蠻和殘忍？你知道嗎，在阿拉伯城市哥多華，有著兩英哩的公共照明，而當時倫敦還只是一個小村莊。」

  • 赖祐萱·人物：康熙停播三年，那些北上的台湾艺人还好吗？
    「对于曾经的康熙粉来说，没有康熙的日子里，整个台湾综艺圈失联了，我们有太多疑问想要解答，就像网上流传已久的那份清单——小甜甜有男朋友了吗？还有人敢对赵哥唱蛤蟆之歌吗？陈汉典到底在干嘛？黄国伦还瞒着寇乃馨去看电影吗？沈玉琳有发RAP吗？瑶瑶最近还爱哭吗？曲老师说话还是那么聒噪吗？殷琦的眼神还是那么迷离吗？」

    「也许不久的将来，又会有一波年轻人在电影院专门选择配音版，好细细品味美国队长那一口浓郁的东北腔。」

  • 吴呈杰·GQ报道：「头顶桂冠，身披枷锁」：获诺贝尔奖七年后的莫言
    「苏童用八个字形容获奖之后的莫言：『头顶桂冠，身披枷锁。』他说他可以预测其他作家的未来，但对莫言只能是『揣度』，因为『他的境遇与我们太不一样了』。诺奖后七年，莫言从『讲故事的人』成了『被讲述的人』。」

  • 好奇心日报：德先生和赛先生，五四和国家，历史学家徐国琦看到的中国人和美国人 | 访谈录
    「核心观点很简单。中国作为 idea 这个概念是单数，是中国人跟外国人缔造出来的东西。 1912 年以前，中国没有任何民族国家象征，没有国号。中国成为中国的正式名称是 1912 年，叫中华民国。在之前，你说我是中国人吗？还是说我是安徽人？过去大明大清，只有到近代，西方人把门打开了，你才要进入国际社会，所以我是从共有的历史这个角度（探讨）。这是我“共有的历史三部曲”最后一本，就讲这个概念怎么来的，里面涉及西方人、中国人，所以写的难度非常大。」

  • R.seon·上流UpFlow：来台湾一趟，我仿佛被卷进了捉奸现场
